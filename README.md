# News Headline Scraper

## Description

A Python-based web scraper that extracts top news headlines from a public website using the Requests and BeautifulSoup libraries. The extracted headlines are displayed in the terminal and saved to a text file.

## Features

* Fetches webpage content automatically
* Extracts top news headlines
* Displays headlines in the terminal
* Saves headlines to a text file
* Demonstrates basic web scraping techniques

## Technologies Used

* Python 3
* Requests
* BeautifulSoup4
* File Handling

## Project Structure

```text
news-headline-scraper/

├── news_scraper.py
```

## How to Run

### Install Dependencies

```bash
pip install requests beautifulsoup4
```

### Run the Program

```bash
python news_scraper.py
```
